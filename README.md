# Ruffier Test (Python) / Тест Руфье на Python

Простая программа на Python для проведения функциональной пробы Руфье — быстрой оценки работоспособности сердца. Программа проводит пользователя через все этапы теста, рассчитывает индекс Руфье и даёт интерпретацию результата.

## Описание

Тест Руфье (проба Руфье-Диксона) используется для определения уровня физической подготовленности и выявления признаков сердечной недостаточности. Он основан на измерении частоты пульса в трёх состояниях: в покое, сразу после нагрузки и через минуту восстановления.

Этот скрипт автоматизирует расчёт индекса и выдаёт готовую оценку. Вам нужно лишь ввести три значения пульса, измеренных за 15 секунд.

## Как это работает

1. Программа запрашивает пульс за 15 секунд в покое (P1).
2. Даёт инструкцию выполнить 30 приседаний за 45 секунд.
3. Сразу после нагрузки запрашивает пульс за 15 секунд (P2).
4. Ждёт 60 секунд (или просит нажать Enter через минуту) и запрашивает пульс за 15 секунд (P3).
5. Переводит 15-секундные измерения в удары в минуту (умножает на 4).
6. Вычисляет индекс Руфье по формуле:

   **Индекс Руфье = (4 × (P1 + P2 + P3) - 200) / 10**

   где P1, P2, P3 — значения пульса в ударах в минуту.
7. Выводит результат и его интерпретацию по стандартной шкале.
