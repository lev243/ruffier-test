# Ruffier Test (Kivy App) / Тест Руфье (мобильное приложение)

Приложение с графическим интерфейсом на Kivy для проведения функциональной пробы Руфье — быстрой оценки работоспособности сердца. Проводит пользователя через все этапы теста, рассчитывает индекс Руфье и выдаёт интерпретацию с учётом возраста.

## Описание

Тест Руфье (проба Руфье-Диксона) используется для определения уровня физической подготовленности и выявления признаков сердечной недостаточности. Он основан на измерении частоты пульса в трёх состояниях: в покое, сразу после нагрузки и через минуту восстановления.

Это приложение автоматизирует расчёт индекса и выдаёт готовую оценку. Вам нужно лишь ввести имя, возраст и три значения пульса, измеренных за 15 секунд. Встроенные таймеры помогут точно соблюсти интервалы.

## Функциональность

- **Экран ввода данных** — ввод имени и возраста (возраст используется для корректировки оценки).
- **Первый замер пульса** — 15-секундный таймер для подсчёта пульса в покое.
- **Нагрузка** — инструкция выполнить 30 приседаний за 45 секунд.
- **Второй замер пульса** — сразу после нагрузки (15 секунд), затем отдых 30 секунд, затем третий замер (15 секунд).
- **Результат** — отображение индекса Руфье и его интерпретации (с учётом возраста).

## Установка

### Требования

- Python 3.6 или выше
- Библиотека Kivy

## Краткое описание кода

Приложение написано на Python с использованием фреймворка Kivy. Код организован в один главный файл `main.py`, который использует вспомогательные модули `instructions.py` (тексты), `ruffier.py` (расчёты) и `seconds.py` (виджет таймера).

**Глобальные переменные**  
- `age`, `name` – данные пользователя.  
- `p1`, `p2`, `p3` – три замера пульса (за 15 секунд).

**Функция `check_int`**  
Проверяет, можно ли преобразовать строку в целое число; возвращает число или `False`.

**Классы экранов (наследники `Screen`)**  
- `InstrScr` – стартовый экран: поля ввода имени и возраста, кнопка перехода к тесту.  
- `PulseScr` – первый замер пульса (в покое): 15‑секундный таймер, ввод результата, переход к нагрузке.  
- `CheckSits` – экран-инструкция: напоминание сделать 30 приседаний за 45 секунд.  
- `PulseScr2` – два замера после нагрузки:  
  * первый 15‑секундный отсчёт (пульс сразу),  
  * отдых 30 секунд,  
  * второй 15‑секундный отсчёт (пульс через минуту).  
- `Result` – финальный экран: вывод персонализированного результата (имя + оценка).

**Главный класс `HeartCheck`**  
Создаёт `ScreenManager`, добавляет все экраны и запускает приложение.
Вспомогательные модули:  
- `instructions.py` – содержит все текстовые константы (подписи, инструкции).  
- `ruffier.py` – функции `get_ruffier_index` (расчёт индекса) и `get_heart_performance` (интерпретация по возрасту).  
- `seconds.py` – класс `Seconds` (виджет‑таймер с событием `done`).
